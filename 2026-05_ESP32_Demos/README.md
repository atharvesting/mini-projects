# ESP32 Demos

These sketches are some beginner-intermediate projects made for the ESP32-S3 Sense as well as a variety of external components like an OLED screen, camera, and MPU 6050.

1. [OLED MPU CAM Party](OLED_MPU_CAM_Party): This demo integrates the OLED screen, MPU 6050 sensor and the OV3660 camera into a single interactive system. The MPU readings are displayed on the screen and the *party trick* is that the camera takes a picture when the contraption is turned upside down.

2. [OLED Moving Text](OLED_Moving_Text): Extremely simple demo that uses the `Adafruit_SSD1306` library in order to communicate with the OLED screen and print a looping text animation.
